About the Class

This course teaches programming and web development, with grading based on participation and projects.

Tell me more

Course Objective

The overarching goal of this course is to foster a comprehensive understanding and application of fundamental programming concepts through the creation of web-based programs. The course is designed to instill proficiency in both core programming language concepts and essential web development principles.


Programming Language Concepts

In pursuit of our objective, we will delve into the following critical programming language concepts:
Statements: Learning how to construct clear and concise code instructions.

Variables: Grasping the concept of variables and their pivotal role in programming.

Conditionals (If statements): Understanding conditional statements for decision-making within programs.

Loops: Mastery of loop structures to facilitate repetitive tasks.

Methods / Functions: Proficiency in creating and implementing functions for modular code design.


Web Programming Concepts

In parallel with programming language fundamentals, we will also explore the essential concepts of web development:
HTML (Hypertext Markup Language): In-depth comprehension of HTML for structuring web content.

CSS (Cascading Style Sheets): A comprehensive exploration of CSS for precise web layout and stylistic elements.

JavaScript: Proficiency in JavaScript programming, enabling dynamic web functionality.


Tools

To facilitate our learning journey, we will be utilizing the following tool:

Repl.it: A versatile platform for coding and project development.


Grading

There will be no conventional homework assignments or tests in this course. Instead, your performance will be evaluated as follows:

50% Participation: Active engagement in in-class discussions, including both asking and answering questions. This encompasses individual and group participation.

50% Projects: Your assessment will largely be based on the successful completion of in-class projects, culminating in the development of a comprehensive web application project.


Expectations

To ensure your success in this course, we have outlined the following expectations:

Engagement with Lectures: Actively participate by following along with the lectures and experimenting with the techniques elucidated.

Weekly Submissions: Stay on track by submitting weekly assignments promptly.

Seek Clarification: Do not hesitate to seek clarification when any concept remains unclear. Effective communication is integral to your learning experience.

Creativity and Initiative: Embrace your creative spirit and challenge yourself to push boundaries when working on your projects. Innovation is encouraged and appreciated.

By adhering to these guidelines and embracing the course materials, you will not only gain a profound understanding of programming and web development but also equip yourself with valuable skills for the future.
